Fonty pro generování PDF faktur (Unicode / diakritika).

Pro plnou podporu znaků (např. ś, ě, ř) přidejte sem soubor:
  DejaVuSans.ttf

Stažení (volně šiřitelný font, licence):
  https://dejavu-fonts.github.io/Download.html
  nebo např. dejavu-sans-ttf-2.37.zip z SourceForge (DejaVu Fonts).

Po rozbalení zkopírujte DejaVuSans.ttf do této složky (web_app/fonts/).

Pokud zde font není, generátor zkusí systémový font (Windows: Arial;
Linux: často /usr/share/fonts/truetype/dejavu/DejaVuSans.ttf).
