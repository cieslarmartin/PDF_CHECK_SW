Jak Agenta spustit (3 kroky)

1

Stáhněte

Klikněte na tlačítko výše a uložte instalátor (.exe) do počítače.

2

Nainstalujte a spusťte

Spusťte instalátor a dokončete průvodce. Při hlášce Windows zvolte „Více informací“ → „Přesto spustit“, pokud se objeví.

3

Přihlaste se

Po spuštění Agenta se přihlaste e-mailem a heslem nebo API klíčem z licence.

🔒 Vaše PDF data neopouštějí váš počítač. DokuCheck Agent analyzuje metadata lokálně pro maximální ochranu duševního vlastnictví.
